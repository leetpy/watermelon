基本操作
========


propertykeys 支持的数据类型
----------------------------

- DOUBLE
- BYTE
- UUID
- FLOAT
- BLOB
- DATE
- OBJECT
- BOOLEAN
- TEXT
- INT
- LONG


Vertex Id 类型

- AUTOMATIC
- PRIMARY_KEY
- CUSTOMIZE_STRING
- CUSTOMIZE_NUMBER

Edgelabels frequency 类型

- SINGLE
- DEFAULT
- MULTIPLE: 必须指定 sort_keys
